Blog API

Technologies Used: NODE.JS, EXPRESS, MONGODB,VSCODE

  • The API follows a RESTful architecture, allowing for intuitive interactions with the endpoints. The data is organized in a logical manner: users, posts, and comments, each with their respective endpoints.

  • User API: Endpoints for user registration, login, and profile retrieval. User authentication using JWT tokens.

  • Posts API: Create, read, update, and delete posts. Implementing pagination to handle large numbers of posts.

  • Through this project, I honed my skills in Node.js, Express, and MongoDB. I gained a deeper understanding of building scalable APIs, handling user authentication, and implementing effective data validation strategies.
